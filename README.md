# Alien Invasion game

Simple 2-D game written using Python 3 and PyGame.

To start a game left mouse click on 'Start' button.

To quit a game press 'q'.

To move a ship:
- left      -> press 'left'
- right     -> press 'right'

To shoot a bullet press 'space'.
